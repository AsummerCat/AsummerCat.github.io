    发表于 2022-08-18 更新于 2023-12-06 本文字数： 3.6k 阅读时长 ≈ 3 分钟

    scala集合

    1. 分为三大类
      序列Seq,集Set,映射Map,所有集合都扩展自Iterable特质;

    2. 对于几乎所有集合类,Scala都同时提供了可变和不可变的版本,分别位于一下两个包

      不可变集合: 该集合不可修改,每次修改都会返回一个新对象

    可变集合: 可直接在原对象上进行修改,而不会返回新对象;
