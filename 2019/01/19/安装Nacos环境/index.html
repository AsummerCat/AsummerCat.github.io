    预备环境准备

    Nacos 依赖 Java 环境来运行。如果您是从代码开始构建并运行Nacos，还需要为此配置 Maven环境，请确保是在以下版本环境中安装使用:

    1. 64 bit OS，支持 Linux/Unix/Mac/Windows，推荐选用 Linux/Unix/Mac。
    2. 64 bit JDK 1.8+；下载 & 配置。
    3. Maven 3.2.x+；下载 & 配置。
    4. 默认密码: admin password
