HomeBrew安装mysql后，如何配置mysql

发表于 2019-01-14 更新于 2023-12-06 本文字数： 2.4k 阅读时长 ≈ 2 分钟

安装mysql

brew install mysql

可以指定版本

brew install mysql@5.7

[启动]Mysql服务

mysql.server start

关闭

mysql.server stop

mysql配置脚本完成初始化

使用mysql的配置脚本：/usr/local/opt/mysql/bin/mysql_secure_installation //mysql 提供的配置向导

启动这个脚本后，即可根据如下命令提示进行初始化设置

设置开机启动
