Dockerfile

DockerfileжҳҜз”ұдёҖзі»еҲ—е‘Ҫд»Өе’ҢеҸӮж•°жһ„жҲҗзҡ„и„ҡжң¬пјҢиҝҷдәӣе‘Ҫд»Өеә”з”ЁдәҺеҹәзЎҖй•ңеғҸе№¶жңҖз»ҲеҲӣе»әдёҖдёӘж–°зҡ„й•ңеғҸгҖӮе®ғд»¬з®ҖеҢ–дәҶд»ҺеӨҙеҲ°е°ҫзҡ„жөҒзЁӢе№¶жһҒеӨ§зҡ„з®ҖеҢ–дәҶйғЁзҪІе·ҘдҪңгҖӮDockerfileд»ҺFROMе‘Ҫд»ӨејҖе§ӢпјҢзҙ§жҺҘзқҖи·ҹйҡҸиҖ…еҗ„з§Қж–№жі•пјҢе‘Ҫд»Өе’ҢеҸӮж•°гҖӮе…¶дә§еҮәдёәдёҖдёӘж–°зҡ„еҸҜд»Ҙз”ЁдәҺеҲӣе»әе®№еҷЁзҡ„й•ңеғҸгҖӮ

жһ„е»әж–№ејҸ

еҲ¶дҪңdocker image зҡ„ж–№ејҸжңүдёӨз§ҚпјҡдёҖжҳҜдҪҝз”Ё Docker containerпјҢзӣҙжҺҘиҝӣе…Ҙ container е…§йғЁпјҢдҪҝз”Ёе‘Ҫд»ӨжқҘеўһеҲӘж”№пјҢеҶҚеҜјеҮәжҲҗ imageпјӣдәҢжҳҜдҪҝз”Ё DockerfileпјҢе°ҶеўһеҲӘж”№зҡ„еҠЁдҪңж–Үжң¬еҢ–(иӯ¬еҰӮжүҖйңҖиҰҒзҡ„зҺҜеўғгҖҒдҫқиө–гҖҒеҸҳйҮҸзӯү)пјҢеҶҚbuildжҲҗ imageгҖӮDockerfile зҡ„ж–№ејҸе®№жҳ“и°ғж•ҙдёҺз®ЎзҗҶпјҢд№ҹжҳҜйҰ–йҖүжҺЁиҚҗзҡ„ж–№ејҸгҖӮ

Dockerfile иҜӯжі•

Dockerfile е‘Ҫд»Ө

==FROM==

  • FROMжҢҮе®ҡдёҖдёӘеҹәзЎҖй•ңеғҸпјҢ дёҖиҲ¬жғ…еҶөдёӢдёҖдёӘеҸҜз”Ёзҡ„ DockerfileдёҖе®ҡжҳҜ FROM дёәз¬¬дёҖдёӘжҢҮд»ӨгҖӮиҮідәҺimageеҲҷеҸҜд»ҘжҳҜд»»дҪ•еҗҲзҗҶеӯҳеңЁзҡ„imageй•ңеғҸгҖӮ
  • FROM дёҖе®ҡжҳҜйҰ–дёӘйқһжіЁйҮҠжҢҮд»Ө Dockerfile.
  • FROM еҸҜд»ҘеңЁдёҖдёӘ Dockerfile дёӯеҮәзҺ°еӨҡж¬ЎпјҢд»ҘдҫҝдәҺеҲӣе»әж··еҗҲзҡ„imagesгҖӮ
  • еҰӮжһңжІЎжңүжҢҮе®ҡ tag пјҢlatest е°Ҷдјҡиў«жҢҮе®ҡдёәиҰҒдҪҝз”Ёзҡ„еҹәзЎҖй•ңеғҸзүҲжң¬гҖӮ

MAINTAINER

  • иҝҷйҮҢжҳҜз”ЁдәҺжҢҮе®ҡй•ңеғҸеҲ¶дҪңиҖ…зҡ„дҝЎ

RUN

  • RUNе‘Ҫд»Өе°ҶеңЁеҪ“еүҚimageдёӯжү§иЎҢд»»ж„ҸеҗҲжі•е‘Ҫд»Өе№¶жҸҗдәӨжү§иЎҢз»“жһңгҖӮе‘Ҫд»Өжү§иЎҢжҸҗдәӨеҗҺпјҢе°ұдјҡиҮӘеҠЁжү§иЎҢDockerfileдёӯзҡ„дёӢдёҖдёӘжҢҮд»ӨгҖӮ
  • еұӮзә§ RUN жҢҮд»Өе’Ңз”ҹжҲҗжҸҗдәӨжҳҜз¬ҰеҗҲDockerж ёеҝғзҗҶеҝөзҡ„еҒҡжі•гҖӮе®ғе…Ғи®ёеғҸзүҲжң¬жҺ§еҲ¶йӮЈж ·пјҢеңЁд»»ж„ҸдёҖдёӘзӮ№пјҢеҜ№image й•ңеғҸиҝӣиЎҢе®ҡеҲ¶еҢ–жһ„е»әгҖӮ
  • RUN жҢҮд»Өзј“еӯҳдёҚдјҡеңЁдёӢдёӘе‘Ҫд»Өжү§иЎҢж—¶иҮӘеҠЁеӨұж•ҲгҖӮжҜ”еҰӮ RUN apt-get dist-upgrade -y зҡ„зј“еӯҳе°ұеҸҜиғҪиў«з”ЁдәҺдёӢдёҖдёӘжҢҮд»Ө. вҖ“no-cache ж Үеҝ—еҸҜд»Ҙиў«з”ЁдәҺејәеҲ¶еҸ–ж¶Ҳзј“еӯҳдҪҝз”ЁгҖӮ

ENV

  • ENVжҢҮд»ӨеҸҜд»Ҙз”ЁдәҺдёәdockerе®№еҷЁи®ҫзҪ®зҺҜеўғеҸҳйҮҸ
  • ENVи®ҫзҪ®зҡ„зҺҜеўғеҸҳйҮҸпјҢеҸҜд»ҘдҪҝз”Ё docker inspectе‘Ҫд»ӨжқҘжҹҘзңӢгҖӮеҗҢж—¶иҝҳеҸҜд»ҘдҪҝз”Ёdocker run вҖ“env =жқҘдҝ®ж”№зҺҜеўғеҸҳйҮҸгҖӮ

USER

  • USER з”ЁжқҘеҲҮжҚўиҝҗиЎҢеұһдё»иә«д»Ҫзҡ„гҖӮDocker й»ҳи®ӨжҳҜдҪҝз”Ё rootпјҢдҪҶиӢҘдёҚйңҖиҰҒпјҢе»әи®®еҲҮжҚўдҪҝз”ЁиҖ…иә«еҲҶпјҢжҜ•з«ҹ root жқғйҷҗеӨӘеӨ§дәҶпјҢдҪҝз”ЁдёҠжңүе®үе…Ёзҡ„йЈҺйҷ©гҖӮ

WORKDIR

  • WORKDIR з”ЁжқҘеҲҮжҚўе·ҘдҪңзӣ®еҪ•зҡ„гҖӮDocker й»ҳи®Өзҡ„е·ҘдҪңзӣ®еҪ•жҳҜ/пјҢеҸӘжңү RUN иғҪжү§иЎҢ cd е‘Ҫд»ӨеҲҮжҚўзӣ®еҪ•пјҢиҖҢдё”иҝҳеҸӘдҪңз”ЁеңЁеҪ“дёӢдёӢзҡ„ RUNпјҢд№ҹе°ұжҳҜиҜҙжҜҸдёҖдёӘ RUN йғҪжҳҜзӢ¬з«ӢиҝӣиЎҢзҡ„гҖӮеҰӮжһңжғіи®©е…¶д»–жҢҮд»ӨеңЁжҢҮе®ҡзҡ„зӣ®еҪ•дёӢжү§иЎҢпјҢе°ұеҫ—йқ  WORKDIRгҖӮWORKDIR еҠЁдҪңзҡ„зӣ®еҪ•ж”№еҸҳжҳҜжҢҒд№…зҡ„пјҢдёҚз”ЁжҜҸдёӘжҢҮд»ӨеүҚйғҪдҪҝз”ЁдёҖж¬Ў WORKDIRгҖӮ

COPY

  • COPY е°Ҷж–Үд»¶д»Һи·Ҝеҫ„ еӨҚеҲ¶ж·»еҠ еҲ°е®№еҷЁеҶ…йғЁи·Ҝеҫ„ гҖӮ
  • <src> еҝ…йЎ»жҳҜжғіеҜ№дәҺжәҗж–Үд»¶еӨ№зҡ„дёҖдёӘж–Үд»¶жҲ–зӣ®еҪ•пјҢд№ҹеҸҜд»ҘжҳҜдёҖдёӘиҝңзЁӢзҡ„urlпјҢ<dest> жҳҜзӣ®ж Үе®№еҷЁдёӯзҡ„з»қеҜ№и·Ҝеҫ„гҖӮ
  • жүҖжңүзҡ„ж–°ж–Үд»¶е’Ңж–Үд»¶еӨ№йғҪдјҡеҲӣе»әUID е’Ң GID гҖӮдәӢе®һдёҠеҰӮжһң жҳҜдёҖдёӘиҝңзЁӢж–Үд»¶URLпјҢйӮЈд№Ҳзӣ®ж Үж–Үд»¶зҡ„жқғйҷҗе°ҶдјҡжҳҜ600гҖӮ

ADD

  • ADD е°Ҷж–Үд»¶д»Һи·Ҝеҫ„ еӨҚеҲ¶ж·»еҠ еҲ°е®№еҷЁеҶ…йғЁи·Ҝеҫ„ гҖӮ
  • еҝ…йЎ»жҳҜжғіеҜ№дәҺжәҗж–Үд»¶еӨ№зҡ„дёҖдёӘж–Үд»¶жҲ–зӣ®еҪ•пјҢд№ҹеҸҜд»ҘжҳҜдёҖдёӘиҝңзЁӢзҡ„urlгҖӮ жҳҜзӣ®ж Үе®№еҷЁдёӯзҡ„з»қеҜ№и·Ҝеҫ„гҖӮ
  • жүҖжңүзҡ„ж–°ж–Үд»¶е’Ңж–Үд»¶еӨ№йғҪдјҡеҲӣе»әUID е’Ң GIDгҖӮдәӢе®һдёҠеҰӮжһң жҳҜдёҖдёӘиҝңзЁӢж–Үд»¶URLпјҢйӮЈд№Ҳзӣ®ж Үж–Үд»¶зҡ„жқғйҷҗе°ҶдјҡжҳҜ600гҖӮ

VOLUME

  • еҲӣе»әдёҖдёӘеҸҜд»Ҙд»Һжң¬ең°дё»жңәжҲ–е…¶д»–е®№еҷЁжҢӮиҪҪзҡ„жҢӮиҪҪзӮ№пјҢдёҖиҲ¬з”ЁжқҘеӯҳж”ҫж•°жҚ®еә“е’ҢйңҖиҰҒдҝқжҢҒзҡ„ж•°жҚ®зӯүгҖӮ

EXPOSE

  • EXPOSE жҢҮд»ӨжҢҮе®ҡеңЁdockerе…Ғи®ёж—¶жҢҮе®ҡзҡ„з«ҜеҸЈиҝӣиЎҢиҪ¬еҸ‘гҖӮ

CMD

  • Dockerfile.дёӯеҸӘиғҪжңүдёҖдёӘCMDжҢҮд»ӨгҖӮ еҰӮжһңдҪ жҢҮе®ҡдәҶеӨҡдёӘпјҢйӮЈд№ҲжңҖеҗҺдёӘCMDжҢҮд»ӨжҳҜз”ҹж•Ҳзҡ„гҖӮ
  • CMDжҢҮд»Өзҡ„дё»иҰҒдҪңз”ЁжҳҜжҸҗдҫӣй»ҳи®Өзҡ„жү§иЎҢе®№еҷЁгҖӮиҝҷдәӣй»ҳи®ӨеҖјеҸҜд»ҘеҢ…жӢ¬еҸҜжү§иЎҢж–Үд»¶пјҢд№ҹеҸҜд»ҘзңҒз•ҘеҸҜжү§иЎҢж–Үд»¶гҖӮ
  • еҪ“дҪ дҪҝз”ЁshellжҲ–execж јејҸж—¶пјҢ CMD дјҡиҮӘеҠЁжү§иЎҢиҝҷдёӘе‘Ҫд»ӨгҖӮ

ONBUILD

  • ONBUILD зҡ„дҪңз”Ёе°ұжҳҜи®©жҢҮд»Өе»¶иҝҹеҹ·иЎҢпјҢе»¶иҝҹеҲ°дёӢдёҖдёӘдҪҝз”Ё FROM зҡ„ Dockerfile еңЁе»әз«Ӣ image ж—¶жү§иЎҢпјҢеҸӘйҷҗе»¶иҝҹдёҖж¬ЎгҖӮ
  • ONBUILD зҡ„дҪҝз”Ёжғ…жҷҜжҳҜеңЁе»әз«Ӣй•ңеғҸж—¶еҸ–еҫ—жңҖж–°зҡ„жәҗз Ғ (жҗӯй…Қ RUN) дёҺйҷҗе®ҡзі»з»ҹжЎҶжһ¶гҖӮ

ARG

  • ARGжҳҜDocker1.9 зүҲжң¬жүҚж–°еҠ е…Ҙзҡ„жҢҮд»ӨгҖӮ
  • ARG е®ҡд№үзҡ„еҸҳйҮҸеҸӘеңЁе»әз«Ӣ image ж—¶жңүж•ҲпјҢе»әз«Ӣе®ҢжҲҗеҗҺеҸҳйҮҸе°ұеӨұж•Ҳж¶ҲеӨұ

LABEL

  • е®ҡд№үдёҖдёӘ image ж Үзӯҫ OwnerпјҢе№¶иөӢеҖјпјҢе…¶еҖјдёәеҸҳйҮҸ Name зҡ„еҖјгҖӮ(LABEL Owner=$Name )

ENTRYPOINT

  • жҳҜжҢҮе®ҡ Docker image иҝҗиЎҢжҲҗ instance (д№ҹе°ұжҳҜ Docker container) ж—¶пјҢиҰҒжү§иЎҢзҡ„е‘Ҫд»ӨжҲ–иҖ…ж–Үд»¶гҖӮ
